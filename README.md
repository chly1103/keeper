# keeper
keeper for team running!

## 1. 定时周报发送

1. 督促team成员发送周报，否则会发送remind邮件
2. 简化编写成本，通过 **issue template** & **markdown** 语法，专注于周报内容本身
3. 统一管理主题、收件人等信息
4. 归档管理周报信息，追踪周报中的问题
